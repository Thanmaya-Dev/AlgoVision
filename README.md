# AlgoVision

AlgoVision is a simple and interactive sorting algorithm visualizer built using React. It helps users understand how sorting works by showing animations of each step.

## Features

- Visualizes sorting algorithms like Bubble Sort, Insertion Sort, and Selection Sort
- Adjustable speed and number of elements
- Color-coded bars for better understanding
- Clean and responsive design

## How to Use

1. Choose a sorting algorithm.
2. Adjust the speed and array size if needed.
3. Click the "Sort" button to see the animation.

## Technologies Used

- React
- Bootstrap
- JavaScript
